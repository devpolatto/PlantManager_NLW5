## Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/birobirobiro/nlw-05-plantmanager.git
```

Download the dependencies and then run the expo

```bash
# Install the dependencies
$ yarn install

# Start the project
$ expo start
```
